This workflow creates an AI-powered personal assistant using n8n. It integrates Telegram, Gmail, and Google Calendar.
Features:

AI Agent: Uses Google Gemini to understand natural language commands.

Email Management: Can read and send emails via Gmail.

Calendar Management: Checks schedule and creates new events in Google Calendar.

Telegram Interface: You can chat with the assistant directly through a Telegram bot.

How to use: Import the .json file into n8n and configure your own credentials for Telegram, Gmail, and Google Gemini.
